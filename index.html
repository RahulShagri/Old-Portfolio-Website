 Projects

Graduate Project - Performance Analysis of Three-Lobe Bearings

Objectives: To study the hydrodynamic pressure, deformation, and stress distribution in three-lobe bearings and compare combinations of eccentricity ratios and materials.

Methodology: Two-phase Computational Fluid Dynamics (CFD) and Two-way Fluid Structure Interaction (FSI) considering cavitation. System coupling between static structural and fluent modules in Ansys Workbench.

Results: Created a database of 24 combinations of bearings. Pressure, deformation, and stress increases with increase in eccentricity ratio. Steel was found to deform and stress less than bronze and brass.

Software used: Autodesk Fusion 360 for modelling and Ansys Workbench for CFD and FSI.

Planar Truss Element FEM Solver

Description: This solver can be used to analyse planar (2 dimensional) truss element structures made of 1D bar elements connected at various nodes that include movement constraints and 1D forces acting at the nodes.

Solver specifications: It uses the direct stiffness method and assembles the global nodal force vector, global element stiffness matrix, and the global nodal displacement vector and uses the elimination approach on the global relationship after applying all the boundary conditions provided by the user to solve for the unknown nodal displacements.

Languages and Libraries used: Purely written in Python, GUI designed on DearPyGUI, matrices assembled and solved using NumPy library, and sketches are imported from FreeCAD.

Check out the source code here.

1 Dimensional Beam Element with 2 Degrees of Freedom FEM Solver

Description: This solver can be used to analyse 2 degrees of freedom of 1 dimensional problems with beam elements connected in series and having 1D forces acting perpendicular to the axis of the beam.

Solver specifications: Uses the direct stiffness method and assembles the global nodal force vector, global element stiffness matrix, and the global nodal displacement vector and uses the elimination approach on the global relationship after applying all the boundary conditions provided by the user to solve for the unknown nodal displacements and slopes.

Languages and Libraries used: Purely written in Python, GUI designed on DearPyGUI, and matrices assembled and solved using NumPy library.

Interactive Assembly Line Balancer

Objective: A python based GUI built using Dear PyGui Python GUI Framework to add multiple tasks and link them together to create and balance an assembly line by assigning tasks to various workstations to achieve the desired output rate with the smallest number of workstations according to the longest or shortest work element rule. This helps create well-balanced workloads for each workstation and minimizes bottlenecks.

Features: Create assembly lines with multiple entry or exit points, adjust tasks times, configure tasks after adding, save and open assembly lines, create a precedence table, enter cycle time and choose from two priority rules, solve to assign all tasks to various workstations to calculate the most efficient assembly line.

Languages and Libraries used: Purely written in Python. GUI built using Dear PyGui.
